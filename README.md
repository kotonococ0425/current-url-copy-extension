# Current URL Copier

現在開いているタブの URL を `Shift+Command+C` でクリップボードへコピーする Chrome 拡張です。

URL は外部に送信しません。現在のタブ URL を取得し、ローカルのクリップボードへコピーするだけです。

## インストール

1. Chrome で `chrome://extensions` を開く。
2. 右上の「デベロッパー モード」をオンにする。
3. 「パッケージ化されていない拡張機能を読み込む」を押す。
4. このフォルダを選択する。

## ショートカット

既定ショートカット:

- macOS: `Command+Shift+C`
- Windows/Linux: `Ctrl+Shift+C`

Chrome や別の拡張と衝突する場合は、`chrome://extensions/shortcuts` を開いて「Current URL Copier」のショートカットを割り当ててください。

## 使い方

コピーしたいページを開いた状態で `Shift+Command+C` を押します。拡張アイコンにも同じコピー動作を割り当てています。

コピーに成功するとページ上部に `URLをコピーしました` と表示され、拡張アイコンに `OK` が短く表示されます。失敗すると拡張アイコンに `ERR` が表示されます。

読み込み済みの拡張を更新した場合は、`chrome://extensions` で「Current URL Copier」の再読み込みボタンを押してください。

## 権限

この拡張は次の権限を使います。

- `tabs`: 現在アクティブなタブの URL を取得するため。
- `activeTab`: ユーザー操作があった現在タブ上でコピー補助処理を実行するため。
- `scripting`: コピー成功メッセージの表示と、クリップボードコピーのフォールバックを現在タブへ注入するため。
- `clipboardWrite`: URL をクリップボードへ書き込むため。
- `offscreen`: Manifest V3 の service worker から安定してクリップボードへ書き込むため。

## プライバシー

- 閲覧履歴を収集しません。
- URL やページ内容を外部サーバーへ送信しません。
- 保存領域を使いません。

## 制限

`chrome://` ページ、Chrome ウェブストア、ブラウザの内部ページなど、Chrome がスクリプト実行を制限しているページでは一部の表示やコピー処理が動かない場合があります。
